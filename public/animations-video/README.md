# Menu icon animations

Short transparent WebM clips (VP9 + alpha) with WebP poster frames, shown on
the main-menu tiles and played once when a tile is tapped.

Rendered offline from `public/animations-shared/` (git-ignored source GLBs)
via headless three.js + ffmpeg. To regenerate or add clips, ask — the render
harness lives outside the repo.

- `boxing`, `taunt`, `warming-up`, `drop-kick` — one `.webm` + `.webp` each
